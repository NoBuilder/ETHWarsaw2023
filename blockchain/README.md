# ETHWarsaw2023 Blockchain

## Development

### Install dependencies

```
npm install
```

### Run linting

```
npm run lint
```

### Run tests

```
npm run test
```

### Run coverage

```
npm run coverage
```

## Deployment

Pass environment variables via `.env` file or shell using the below example.

```ini
ETHERSCAN_API_KEY=abc
GOERLI_URL=https://eth-ropsten.alchemyapi.io/v2/<ALCHEMY_KEY>
PRIVATE_KEY=0x123
```
